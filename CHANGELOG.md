# Changelog

*Sorted is an extension that makes organizing bookmarks easier. It is designed to fit inside the existing bookmarks model and stay out of the user's way.  
Sorted has zero configuration and zero background processing. It will only work when the user clicks on the extension's icon.*

## [2.7.0] - 14 March 2020
### What's new
- The `add` button has been removed. While adding a new item, users can now press the `Enter` key for the same effect.
- While adding a new item, a check for duplicate keywords is also performed and duplicates are automatically removed.
- A lot of UI improvements *(Checked some margins and swapped some padding)*.
